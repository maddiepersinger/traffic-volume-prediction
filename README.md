# traffic-volume-prediction
Predicting traffic volume in Georgia using R and machine learning models

**Author:** [Madeline Persinger]  
**Tools:** R, SQL  
**Project Type:** Data Science / Machine Learning

---

## Project Overview
This project predicts traffic volume in :contentReference[oaicite:0]{index=0} using real-world transportation data. The goal is to build, compare, and evaluate multiple predictive models to identify patterns and forecast traffic levels.

---

## Objectives
- Collect and clean traffic data for analysis  
- Explore and understand trends and patterns in the dataset  
- Build regression and classification models to predict traffic volume  
- Compare model performance to select the best predictive approach  
- Present insights using visualizations and structured reports  

---

## Data
- The dataset includes traffic counts, timestamps, and location information for multiple sites across Georgia.  
- Data cleaning involved handling missing values, feature engineering, and data formatting.  

---

## Models Used
- **Regression:** Linear Regression  
- **Classification:** Logistic Regression, K-Nearest Neighbors (K-NN), Naive Bayes, Support Vector Machines (SVM)  
- **Tree-based:** Decision Trees  
- **Statistical Learning:** Linear Discriminant Analysis (LDA), Quadratic Discriminant Analysis (QDA)
